# Secure_Banking_Interface
This project is a console-based banking application developed using Java, demonstrating core Object-Oriented Programming concepts such as encapsulation, abstraction, and modular design. It allows users to perform essential banking operations including account creation, deposit, withdrawal, balance inquiry, and transaction history tracking. The system incorporates a PIN-based authentication mechanism to ensure secure user access and protect sensitive data. Data structures like HashMap and ArrayList are used for efficient account management and transaction storage, making the application scalable and easy to maintain, while also reflecting real-world banking system functionality.
