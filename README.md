# Secure_Banking_Interface
A console-based banking application developed using Java, demonstrating core Object-Oriented Programming (OOP) concepts. The system provides secure banking operations with PIN-based authentication.

## Features
- Account creation and management
- Deposit and withdrawal operations
- Balance inquiry
- Transaction history tracking
- Secure PIN-based user authentication

## Technologies Used
- Java
- OOP Concepts (Encapsulation, Abstraction, Modularity)
- Data Structures (HashMap, ArrayList)

## Security
- PIN-based authentication for user verification
- Encapsulation to protect sensitive data (PIN, balance)

##  Project Structure
- `Account.java` → Handles account operations
- `Bank.java` → Manages multiple accounts
- `Main.java` → Controls application flow and user interaction

## How to Run
1. Clone the repository
2. Open in any Java IDE (Eclipse / IntelliJ / VS Code)
3. Compile and run:
   ```bash
   javac Main.java
   java Main
